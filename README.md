Python 停車管理計費系統 (V1.0)
這是一個基於 Python 開發的簡易停車場管理系統，具備車牌格式驗證、即時時區紀錄與智慧計費邏輯（含 10 分鐘寬限期）。

 模組架構
本專案採用模組化設計，拆分為三個核心檔案：

main.py：系統主程式，負責流程控制、資料儲存（使用 dict）與費用計算。

car_module.py：車牌驗證模組，負責檢查 XXX-XXXX 格式。

datime_test.py：時間管理模組，提供 Asia/Taipei 精準時區物件。

 核心功能

時區追蹤：採用 ZoneInfo 確保與台灣標準時間同步。

復仇計費邏輯：

10 分鐘內免費：針對短暫停放車輛不予計費。

無條件進位制：超過 10 分鐘後，採每小時為單位計費（不滿一小時以一小時計算）。
